# Habilidades de Agente (Planner, Generator, Healer)

## Planner
Cuando te pida planificar, explora el contexto actual, identifica los flujos de usuario clave y crea un plan de pruebas en Markdown detallando pasos, selectores y resultados esperados.

## Generator
Utiliza el plan generado para crear archivos de prueba (.spec.ts) utilizando la sintaxis de Playwright v1.56+.

## Healer
Si una prueba falla, analiza el error y el código, y repara automáticamente el selector o la lógica.
